# 📊 Michael's Market Dashboard

I built this because I wanted to learn Python and finance at the same time. I got tired of switching between tabs to track stocks. It fetches live data from Yahoo Finance and visualizes price trends, moving averages, daily returns, and volatility.

## 🚀 Features
- Multi-stock comparison (e.g., Apple, Tesla, S&P 500, etc.)
- Adjustable time ranges (1 month to 5 years)
- Moving average overlays
- Daily return analysis
- Summary statistics (mean return, volatility)

## 🛠️ Tech Stack
- **Python** (pandas, matplotlib, yfinance)
- **Streamlit** (for web dashboard)
- **Deployed on Streamlit Cloud**
